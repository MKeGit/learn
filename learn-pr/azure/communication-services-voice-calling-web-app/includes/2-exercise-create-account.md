You can use Azure Communication Services to make calls programmatically. In this unit, you create a Communication Services resource to use.

## Create a resource

> [!IMPORTANT]
> Activities in this module will result in a billed charge. Be sure to perform the cleanup steps in the final module to avoid recurring charges, even if you don't finish the whole module.

1. Go to [the Azure portal](https://portal.azure.com/?azure-portal=true).
1. On the Azure portal menu or from the **Home** page, select **Create a resource**.
1. In the **Search services and marketplace** box or the Azure search bar, enter *Communication*.
1. Select **Communication Services** in the results, and then select **Create**.
1. Specify your subscription, resource group (including the location if creating a new resource group), name, and data location of your Communication Services resource.
1. Select **Review + Create**, and then select **Create**.
1. After Azure finishes the deployment of your resource, select **Go to resource**.
