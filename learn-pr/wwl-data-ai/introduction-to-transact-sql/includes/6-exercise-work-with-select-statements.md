Now it's your chance to try the Transact-SQL techniques you've learned about for yourself.

You can complete this exercise using one of the following options:

- Microsoft SQL Server
- Azure SQL Database

::: zone pivot="sql-server"

A virtual machine containing the client tools you need is provided, along with the exercise instructions. Use the button above to launch the virtual machine.

> [!NOTE]
> A limited number of concurrent sessions are available - if the hosted environment is unavailable, try again later.

Alternatively, you can use these [setup instructions](https://go.microsoft.com/fwlink/?linkid=2294500&azure-portal=true) to install Microsoft SQL Server and create a sample database, and follow these [exercise instructions](https://go.microsoft.com/fwlink/?linkid=2293251&azure-portal=true).

::: zone-end

::: zone pivot="azure-sql-database"

> [!NOTE]
> To use Azure SQL Database, you will need a [Microsoft Azure subscription](https://azure.microsoft.com/free?azure-portal=true) in which you have administrative access.

To set up a database for this exercise, sign into your Azure subscription and follow the [setup instructions](https://go.microsoft.com/fwlink/?linkid=2294301&azure-portal=true) to provision Azure SQL Database.

Launch the exercise and follow the instructions.
