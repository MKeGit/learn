In this exercise, you configure logical replication, which is native to PostgreSQL. You create two servers, which act as publisher and subscriber. Data in the zoodb replicates between them.

> [!NOTE]
> You need your own Azure subscription to complete this exercise. If you don't have an Azure subscription, you can create an [Azure free trial](https://azure.microsoft.com/free).
